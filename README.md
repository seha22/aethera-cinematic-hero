# Aethera Cinematic Hero

A beautiful fullscreen cinematic hero section built with React + Vite + Tailwind CSS + TypeScript.

## Features
- Custom looping video background with fade in/out
- Instrument Serif + Inter fonts
- Smooth animations
- Fully responsive

## Getting Started

```bash
npm install
npm run dev
```

Live demo: https://seha22.github.io/aethera-cinematic-hero
